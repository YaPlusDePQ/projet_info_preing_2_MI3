-f <file> or -f <file directory> : to specify the name of the original file or its directory  
-h : height
-m : moisuture 
-w : wind 
-p : pression (you have to add space after the p )
    -p 1: sort average, min and max pressures according to the stations' IDs 
    -p 2: sort average pressures per date/hour according to the stations' IDs 
    -p 3: sort average pressures per date/hour according to the stations' IDs in a chronological order
if the mode is different from 1,2 or 3 it will exit (2)
-t : temperature (you have to add space after the t )
    -t 1: sort average, min and max temperatures according to the stations' IDs 
    -t 2: sort average temperature per date/hour according to the stations' IDs 
    -t 3: sort average temperatures per date/hour according to the stations' IDs in a chronological order
if the mode is different from 1,2 or 3 it will exit (2)
-d <min> <max>: date 
-r : to sort in a reverse order
-F: France & Corse
-A: West Indies
-G: French Gyana
-O: Indian Ocean
-S : Saint-Pierre et miquelon 
-Q : Antartic
--avl : sort data using avl
--abr : sort data using abr
--tab : sort data using tab 
--help : display the manual
